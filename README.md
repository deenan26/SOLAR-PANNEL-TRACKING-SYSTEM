# Solar Panel Tracking System

## 📌 Project Overview
This project implements an automatic solar panel tracking system designed to maximize solar energy generation by continuously aligning the solar panel with the sun's position.

The system uses Light Dependent Resistors (LDRs) to detect sunlight intensity from different directions. An Arduino Uno processes the sensor readings and controls servo motors to adjust the panel orientation accordingly.

By tracking the sun’s movement throughout the day, the system ensures optimal sunlight exposure and improves the efficiency of solar energy harvesting compared to fixed solar panels.

## 🔧 Working Principle
The system uses two or more LDR sensors placed at different positions on the solar panel. These sensors detect the intensity of sunlight falling on each side.

The Arduino Uno continuously reads the analog values from the LDR sensors. If one sensor receives more light than the other, the Arduino commands the servo motor to rotate the solar panel toward the direction of higher light intensity.

This automatic adjustment allows the solar panel to follow the sun throughout the day, ensuring maximum energy absorption.
